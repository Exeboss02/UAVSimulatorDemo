# Build Instructions

Make sure you have Visual Studio 2026 with cmake 3.18+ installed and "Visual Studio 18 2026" generator downloaded (Should be included)
Cmake can be added to Visual Studio 2026 by adding "Desktop Development with C++" to Visual Studio in the installer.

run "./build release" in the Visual Studio terminal from the root directory

# Running the project
run "./run release" in the Visual Studio terminal from the root directory
